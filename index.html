What is the BayouBot?

The BayouBot is an inexpensive robot designed with education in mind that is currently under development. The hardware of the BayouBot allows for simple educational robotics projects across many grade levels, and can even be used for more advanced projects, such as in an introductory robotics course. A programming workspace is also in development for the BayouBot that will have a similar feel to MIT's Scratch. This will allow younger children to program the BayouBot using a very simple, graphical language that teaches them the basic concepts of programming. A Java API also exists for more advanced users that may want more control over the project, or that may want to teach some aspects of Java or more advanced robotics using the BayouBot.
